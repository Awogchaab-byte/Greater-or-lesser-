# 🔢 Number Comparison Program (C++)

This C++ program takes **two integer inputs** from the user and compares them to determine which one is greater.

---

## 🧠 Objective
To practice conditional statements and relational operators in C++.

---

## 📜 Problem Statement
> Write a C++ program that reads two integers and prints **"X is greater"** if the first number is larger, otherwise prints **"Y is greater"**.

---

## 💻 Code Overview
The program:
1. Prompts the user to enter two integers (`x` and `y`).
2. Uses the **greater-than (`>`)** operator to compare them:
   ```cpp
   if (x > y)
       cout << "X is greater";
   else
       cout << "Y is greater";
